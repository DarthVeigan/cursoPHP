- Clonar pastas - git clone link nomeDaNovaPasta
- Criar novo repositório - cria a nova pasta, acessa a pasta com o CMD e digita git init. 
- Para saber se foi criado, acesse o caminho da pasta.\git

- git add nome.ext - Leva para uma sala de espera ( stage area ) Vai aguardar alterações do commit!
- git add --all - Vai por todos os arquivos na stage area
- git commit -m " Por que fez a alteração "
- git push - Empurrar arquivos pro servidor remoto!
